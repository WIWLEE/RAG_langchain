#### 실행방법
1. 가상환경 생성
conda create -n RAG python==3.11.0

2. 가상환경 활성화
conda activate RAG

3. Requirement 다운로드
pip install -r requirements.txt

4. .env 생성 및 설정
OPENAI_API_KEY와 TAVILY_API_KEY를 발급받아 .env 파일에 작성
(OPENAI_API_KEY는 유료 결제 필요)

5. RAG_implementation.ipynb 실행
